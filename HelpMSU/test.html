1. Файл Excel должен содержать колонки:
    Номер накладной, Дата формирования накладной, Артикул поставщика, Наименование, Цена, Количество, Ставка НДС, ШтрихКод, ГТД, ОВХ, Кратность, Номер счёта и прочее....


  2. Этот файл должен подгружаться в 1С без внесения изменений в нашу наменклатуру



  3. Нужна возможность выбрать поставщика и счета из списка ожидающихся, необработаных, комерческих предложений



  4. После выбора контрагента и счетов, позиции из файла должны сооставляться с позициями из счетов в 1С



  5. Если есть расхождения, то их список должен выгружаться для менеджеров/координаторов для последующей корректировки счетов.



  6. После всех корректировок должно формироваться поступление с номером и датой из файла Excel



  7. На основании подтянутых счетов должны формироваться заявки на сборку.
