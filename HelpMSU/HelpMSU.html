  4. Внесение трудозатрат фасовщиц:

    1. Открыть список производств.
    2. Найти нужное производство.
    3. Нажать правую кнопку мыши и в контекстном меню выбрать "подчинённые документы".
    4. В выпавшем списке выбрать Карточку учёта производительности и открыть её.
    5. В карточку добавить фасовщиц. Для этого нажать кнопку справа:
      для добавления одного человека
      для добавления нескольких человек
    6. В колонке выбрать фасовщиц в соответствии с карточками:
      Для этого нужно левой кнопкой щёлкнуть в соответствующей строке и из выпавшего меню выбрать нужню фамилию.
      Для изменения количества в меньшую сторону на вкладке "общая таблица" дважды щёлкнуть левой кнопкой мыши по количесву, которое нужно уменьшить и в выпавшем окне ввести нужное количество.
      Для изменения количества в большую сторону указать фасовщика, затем перейти на вкладку с фамилией фасовщика, дважды щёлкнуть левой кнопкой мыши по количесву, которое нужно увеличить и в выпавшем окне ввести количество.
    7. При необходимости внести ОВХ "Обьёмно-Весовые Характеристики" в соответствующие поля карточки. После чего нажать кнопку .
      ВНИМАНИЕ!
      Нужно отслеживать дату карточки таким образом, чтобы она попадала в период, за который предстоит выплата.
